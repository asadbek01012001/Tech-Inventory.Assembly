## 🎨 Design Elements

### Color Scheme
```
Primary:   Blue (#2563eb, #1e40af)
Success:   Green (#16a34a, #15803d)
Warning:   Yellow (#eab308)
Danger:    Red (#dc2626, #b91c1c)
Neutral:   Gray (#1f2937 to #f3f4f6)
```

### Typography
```
Headers:    text-2xl, text-3xl, text-4xl (bold)
Labels:     text-base (semibold)
Body:       text-base, text-lg (medium)
Small:      text-sm
```

### Spacing
```
Padding:    px-4, py-3, p-6, p-8 (katta)
Gap:        gap-4, gap-6 (katta)
Margin:     mb-6, mb-8 (katta)
Rounded:    rounded-lg (8px)
```
